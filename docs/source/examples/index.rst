Examples
========

There are three types of examples. Under **Basics** you'll find tips and tricks for situations commonly encountered when working with the package.
**Snippets** are short usage examples that demonstrate a specific function of *thebeat*.
Finally, in **Use case scenarios** we recreate existing research but then use *thebeat* for generating the
stimuli or analyzing the data.
