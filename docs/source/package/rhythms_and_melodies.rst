Rhythms and melodies
====================

There are two classes that make up the core of this module:

* The :py:class:`Rhythm<thebeat.rhythm.Rhythm>` class, which contains a metrical, **rhythmical sequence** (with a time signature etc.).
* The :py:class:`Melody<thebeat.melody.Melody>` class, which contains a **rhythm as well as pitch information**.
